低配版 PvZ
僵尸从地图上方随机刷新，每回合向玩家移动一格，玩家初始有一个植物，每回合可以种植物，移动植物，合并两个攻击力相同的植物从而升级植物，打败一个僵尸随机掉落植物。
具体操作为选择要进行的操作，然后依次根据提示输入 src_row, src_col, dst_row, dst_col。

e.g. 从植物储备中将第一列的储备植物种植到种植区的第二行第三列
输入 1 2 3

项目运行说明：直接运行 main.exe
